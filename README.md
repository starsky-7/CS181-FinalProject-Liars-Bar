# CS181-FinalProject-Liars-Bar

This project is a framework for playing the Liars Bar game with reinforcement learning, expectimax and heruistic search.

## State Space
The state space consists of the following components:
1. The current player's turn
2. The current number of players
3. The current number of cards in the deck
4. The current number of different types of cards each player has
5. The history of actions taken by each player
